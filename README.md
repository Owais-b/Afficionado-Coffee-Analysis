# ☕ Sales Trend and Time-Based Performance Analysis for Afficionado Coffee Roasters

## 📌 Project Overview
This project analyzes transaction-level sales data for Afficionado Coffee Roasters to uncover temporal demand patterns and support data-driven business decisions. The analysis focuses on understanding sales trends, peak transaction hours, store performance, and product demand using Python and Streamlit.

---

## 🎯 Objectives

### Primary Objectives
- Identify overall sales trends across the dataset.
- Determine peak and off-peak transaction hours.
- Analyze sales performance across different store locations.

### Secondary Objectives
- Compare temporal demand patterns across stores.
- Support staffing and operational planning.
- Generate evidence-based business recommendations.

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Owais-b/Afficionado-Coffee-Analysis.git
cd Afficionado-Coffee-Analysis
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Streamlit Dashboard

```bash
streamlit run dashboard/dashboard.py
```
